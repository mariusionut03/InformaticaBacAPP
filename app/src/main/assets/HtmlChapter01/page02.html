Cuvant inainte

Infromatica este o stiinta reala, deci, pentru note cat mai mari, exercitiul este esential.

Nu veti lua nota maxima doar prin citirea teoriei, dupa fiecare capitol este recomandata aprofundarea prin scrierea unor programe in aplicatii precum Code::Blocks, Visual Studio etc.

Verifica-ti cunostintele singur si priveste aplicatia doar ca pe un memorator.

Textul scris intr-o casuta ca cea de mai jos reprezinta cateva linii de cod. Cateodata textul este prea lung, asa ca va trebui sa rulati spre dreapta. Testati mai jos si spor la invatat! Progresul ti se va salva, deci inchide aplicatia si continua mai tarziu fara nicio problema.
