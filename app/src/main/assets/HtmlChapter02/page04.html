2. Structura repetitiva

a. Instructiunea PENTRU

// Deruleaza spre dreapta pentru a vedea codul complet
┌ pentru <variabila> ← <expresie initiala>, <expresie finala> , <pas> executa
│ <instructiuni>
└■

In locul campurilor 'expresie initiala' si 'expresie finala' se introduc doua numere intregi sau variabile.
In locul campului 'pas' se va scrie o expresie (Exemplu: i = i + 1).
Variabila va primi valoarea expresiei initiale si va aplica expresia 'pas' dupa fiecare executare a instructiunilor, pana cand valoarea ei va ajunge egala cu cea finala.

Exemplu. Suma primelor n numere naturale:

S ← 0
citeste n (numar natural)
┌ pentru i ← 1, n executa
│ S ← S + i
└■
scrie S


b. Instructiunea CAT TIMP ... EXECUTA

┌ cat timp <conditie> executa
│ <instructiuni>
└■

Se verifica conditia si executa instructiunile pana cand ea se indeplineste.


c. Instructiunea EXECUTA ... CAT TIMP

┌ executa
│ <instructiuni>
└■ cat timp <conditie>

Se executa instructiunile o singura data, dupa care se verifica conditia si se repeta procesul pana cand ea se indeplineste.


d. Instructiunea REPETA ... PANA CAND

┌ repeta
│ <instructiuni>
└■ pana cand <conditie>

Se executa instructiunile, se verifica conditia si, daca este FALSA, se executa instructiunile pana cand ea devine ADEVARATA.
