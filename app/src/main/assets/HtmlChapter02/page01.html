Pseudocod

Algoritmul este un set finit de pasi bine pusi la punct, scopul lui fiind rezolvarea unei probleme sau al unei categorii de probleme.

Pseudocodul este considerat un limbaj de programare generalizat. Nu poate fi citit si executat de calculator, dar este inteles de orice programator, indiferent de limbajul de programare pe care il foloseste.

Un exemplu de cod in acest limbaj de programare:

// Citirea unui numar de la tastatura
citeste x (numar intreg)

// Atribuirea unei valori lui x
x ← 3

// Afisarea valorii retinute in variabila x
scrie x

// Folosind aceste 3 instructiuni, putem
calcula media aritmetica a doua numere:
citeste x, y (numere intregi)
suma ← x + y
media ← suma / 2
scrie media
// Daca citim valorile x = 3 si y = 7, se va afisa
la final numarul 5.
