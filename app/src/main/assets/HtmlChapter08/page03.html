Transmiterea parametrilor

Am vazut pe paginile anterioare cum putem transmite parametri catre subprograme.

#include <iostream;
using namespace std;
int cmmdc(int x, int y)
{
while(x != y)
if(x > y)
x -= y;
else
y -= x;
return x;
}

int main()
{
int a, b;
cin >> a >> b; // Introducem: 12 18
cout << a << ' ' << b << endl; // Afiseaza: 12 18
cout << cmmdc(a,b) << endl; // Afiseaza 6
cout << a << ' ' << b; // Afiseaza 12 18
return 0;
}

In programul de mai sus, dam valorile 12 si 18 pentru a si b. (a=12, b=18).

Dupa chemarea subprogramului 'cmmdc' cu parametrii a si b, valoarea lor nu se schimba dupa rularea comenzilor din subprogram. Cand afisam cele doua variabile la final, valorile lor raman tot 12 si 18.

int cmmdc(int x, int y)
{
while(x != y)
if(x > y)
x -= y;
else
y -= x;
return x;
}

int main()
{
cout << cmmdc(2, 6) << endl; // Afiseaza 2
cout << cmmdc(13, 7) << endl; // Afiseaza 1
cout << cmmdc(12, 36) << endl; // Afiseaza 12
cout << cmmdc(26, 39) << endl; // Afiseaza 13
cout << cmmdc(8.5 , 12.56) << endl; // Afiseaza 4
return 0;
}

Putem chema subprogramul de cate ori vrem noi. Parametrii sunt de tip 'int', deci valorile '8.5' si '12.56' din ultima chemare, vor fi convertite automat in '8' si '12'.
