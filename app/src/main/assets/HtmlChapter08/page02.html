Structura unei functii

O sa rescriem functia de pe pagina anterioara si o sa explicam ce inseamna fiecare lucru:

int subprogram(int a, int b)
{
int c = a + b;
return c;
}

1. Observam ca functia este declarata cu 'int'(numar intreg). Acesta reprezinta tipul de datÄƒ ce este returnata si poate fi inlocuit cu majoriatatea tipurilor de date pe care le cunoastem('char', 'double', 'float' etc.).

Daca nu vrem sa returnam nimic, folosim tipul 'void'. Un exemplu este o functie care doar afiseaza ceva pe ecran, ea va fi declarata cu 'void'.

2. 'subprogram' este numele functiei si poate fi editat dupa placul nostru.

3. '(int a, int b)' sunt parametrii functiei, doua numere de tip intreg si trebuie precizate la chemarea functiei. O functie poate fi lipsita de parametri, in cazul nostru am fi definit-o 'int subprogram()', fara niciun text intre paranteze.

4. '{ }' parantezele delimiteaza instructiunile.

5. Scriem instructiunile functiei intre paranteze.

Prototipul unei functii

#include <iostream>;
using namespace std;

int main()
{
g();
return 0;
}

void g()
{
cout << "Salutare!"
}

// Eroare

Acest program va da eroare. Functia 'g()' este chemata in 'main()' inainte ca ea sa fie definita.

Totul se rezolva mentionand functia 'g()' la inceputul programului.

#include <iostream>;
using namespace std;

void g(); // Acest

int main()
{
g();
return 0;
}

void g()
{
cout << "Salutare!"
}
