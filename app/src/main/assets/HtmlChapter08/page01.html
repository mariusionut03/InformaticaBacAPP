Subprograme

Un subprogram este o colecție de tipuri de date, variabile, instrucțiuni care îndeplinesc o anumită sarcină (calcule, citiri, afișări), atunci când este apelat de un program sau de un alt subprogram.

Subprogramele sunt utile in programele unde o serie de instructiuni este repetata de mai multe ori, asa ca o scriem o singura data si, dupa, o chemam de fiecare data cand avem nevoie.

Pentru a intelege conceptul, cel mai bine dam un exemplu in C++:

#include <iostream>;
using namespace std;

int subprogram(int a, int b)
{
int c = a + b;
return c;
}

int main()
{
int m, n;
cin >> m >> n;
cout << subprogram(m, n);
return 0;
}

In codul de mai sus avem definita inca o functie pe langa 'main', numita 'subprogram'. Ea primeste 2 valori, retinute cu numele 'a' si 'b'. La apelarea functiei 'subprogram', se va face suma celor doua numere ce se va retine in variabila 'c', urmand sa fie returnata inapoi in main. Observam ca apelarea functiei se face in comanda 'cout', asa ca valoarea returnata va fi afisata pe ecran.
