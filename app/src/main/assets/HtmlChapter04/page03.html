Structuri alternative

În anumite situații, este necesară executarea unor instrucțiuni în cadrul unui program numai în anumite condiții. Structurile alternative (numite și structuri de decizie) permit rezolvarea unor asemenea situații.

Instrucțiunea if

Instrucțiunea if este cea mai utilizată structură alternativă. Ea are mai multe forme:

if ( Expresie )
  Instrucțiune1;

Varianta de mai sus verifica expresia, iar daca ea este adevarata executa Instructiune1. In caz contrar, nu face nimic.

if ( Expresie )
  Instrucțiune1
else
  Instrucțiune2

Varianta de mai sus verifica expresia. Daca este adevarata, executa Instructiune1, in caz contrar va executa Instructiune2.

if ( Expresie1 )
  Instrucțiune1
else if ( Expresie2 )
  Instrucțiune2
else if ( Expresie3 )
  Instructiune3
.
.
.
else if ( ExpresieN )
  InstructiuneN;
else
  InstructiuneM;

Putem adauga cat de multe expresii vrem, daca adaugam instructiunea else if, ca in exemplul de mai sus. Programul va verifica pe rand fiecare expresie. Prima care este adevarata, i se va executa instructiunea. Daca doua expresii sunt adevarate, doar prima gasita se va lua in considerare.

Exemple:

int x;
cin >> x;
if(x % 2 == 0)
  cout << x << " este par";
else
  cout << x << " este impar";

sau, un exemplu cu instructiuni compuse:

int x, y, dif;
cin >> x >> y;
if(x > y)
{
  dif = x - y;
  cout << "Diferenta este " << dif << ".";
}
else if (x < y)
{
  dif = y - x;
  cout << "Diferenta este " << dif << ".";
}
else
{
  cout << 0;
}

Instructiunea switch

Instrucțiunea switch permite executarea unor instrucțiuni, în funcție de egalitatea unei expresii cu anumite valori numerice constante:

switch ( Expresie )
{
case Constanta_1:
  Grup_Instructiuni_1
  break;
case Constanta_2:
  Grup_Instructiuni_2
  break;
…
case Constanta_N:
  Grup_Instructiuni_N
  break;
default:
  Grup_Instructiuni_default
  break;
}

Mod de executie

1: Se evaluează Expresie
2.1: Dacă valoarea expresiei este egală cu una dintre valorile constante din clauzele case, se execută instrucțiunile din grupul de instrucțiuni corespunzător, apoi se trece la instrucțiunea de după switch
2.2: Dacă valoarea expresiei nu este egală cu niciuna dintre valorile constante din clauzele case, se verifică existența clausei default;
3.1: Dacă există clauza default, se execută instrucțiunile din grupul de instrucțiuni corespunzător clauzei default, apoi se trece la instrucțiunea de după switch
3.2: Dacă nu există clauza default, se trece la instrucțiunea de după switch

Exemplu:

#include <iostream>
using namespace std;
int main()
{
  int zi;
  cin >> zi;
  switch(zi)
  {
  case 1:
    cout << "Luni\n"; break;
  case 2:
    cout << "Marti\n"; break;
  case 3:
    cout << "Miercuri\n"; break;
  case 4:
    cout << "Joi\n"; break;
  case 5:
    cout << "Vineri\n"; break;
  case 6:
  case 7:
    cout << "WEEKEND!!!\n"; break;
  default:
    cout << "Numarul zilei este incorect\n"; break;
  }
  return 0;
}

Observați mai sus că în clauza case 6: nu avem instrucțiuni. Dacă variabila zi are valoarea 6 sau 7 se va afișa "WEEKEND!!!"
