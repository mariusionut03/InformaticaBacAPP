Tablouri unidimensionale

Tablourile unidimensionale sunt declarate astfel:

tipDeBazÄƒ denumire[Dimensiune];
de exemplu: int X[4];

Ne putem imagina tablourile unidimensionale ca un sir de valori.
Dimensiunea tabloului declarat mai sus, adica X[4], este 4.
Acest tablou poate lua 10 valori numerotate de la 0 la 3.

Element 17 23 5 0
Indice 0 1 2 3

Accesarea unui element se face prin operatorul de indexare [].

cout << X[0]; // Afiseaza 17
cout << X[3]; // Afiseaza 0
cout << X[4]; // Eroare

Scrierea unui tablou intr-un program

int main()
{
int A[100]; // Declaram un tablou cu maxim 100 de elemente
int n; cin >> n; // n reprezinta numarul elementelor pe care le vom citi
for(int i = 0; i < n; i++)
cin >> A[n];

Programul o sa citeasca de la tastatura, pe rand, elementele lui A de la 0 la n-1.

Important! n trebuie sa fie mai mic decat dimensiunea lui A, adica 100.

Afisarea unui vector

Important! Nu se poate face afisarea cu "cout << X;". Trebuie specificat indexul. Varianta corecta este:

for(int i = 0; i < n; i++)
cout << X[i] << ' ';

sau in ordine descrescatoare:

for(int i = n-1; i >= 0; i--)
cout << X[i] << ' ';

Initializarea elementelor unui tablou

Declararea unui tablou se poate face specificand si elementele.

int A[] = {10 , 20 , 30 , 40};
int B[10] = {10 , 20 , 30 , 40};
int C[10] = {0};

A va avea patru elemente, cu valorile A[0] = 10, A[1] = 20, A[2] = 30, A[3] = 40.
B va avea zece elemente; primele patru vor avea valorile B[0] = 10, B[1] = 20, B[2] = 30, B[3] = 40, celelalte vor avea valoarea 0!!!
C va avea zece elemente. Toate vor avea valoarea 0!!!
