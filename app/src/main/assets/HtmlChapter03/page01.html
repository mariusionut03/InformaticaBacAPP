Bazele limbajului C++

Limbajul C++ este cel pe care il invatam in Romania pentru bacalaureat. Fata de pseudocod, acesta poate fi citit de calculator.


Acesta este primul program C++. El afiseaza pe ecran textul "Hello world".

#include <iostream>
using namespace std;
int main()
{
  cout << "Hello world";
  return 0;
}
// Afisare: Hello world

Acum o sa luam pe rand fiecare linie pentru a explica programul:

#include <iostream>

Aceasta linie ne introduce in program libraria de comenzi iostream. Aceasta ne lasa sa utilizam cout (Afisarea pe ecran) si cin (Citirea de la tastatura), comenzi esentiale pentru un program. Mai tarziu o sa folosim si alte librarii, cum ar fi cstring (Siruri de caractere) si fstream (Citirea si scrierea in fisiere externe).

using namespace std;

Acesta este un namespace ce ne usureaza treaba. 'std' este prescurtarea de la standard. Un exemplu prin care ne dam seama ca ne ajuta foarte mult, in loc sa scriem direct cout cand folosim aceasta comanda, ar fi trebuit sa scriem std :: cout pentru ca aceasta sa functioneze.

int main()

Aici am declarat o functie. Functia main este necesara in orice program C++. 'int' vine de la numar intreg si reprezinta tipul de data pe care functia o sa o returneze la final. 'main' este numele functiei. O sa vorbim mai amanuntit despre functii la capitolul 'Subprograme'.

{

Aceasta paranteza marcheza inceputul functiei main.

cout << "Hello world";

Comanda 'cout' vine de la 'console output' si cu ajutorul ei afisam text pe ecran. Sagetile '<<' ne arata o directie. Indica faptul ca ce urmeaza dupa ele este inserat in 'cout', deci se va afisa pe ecran. "Hello world" ne arata textul pe care il afisam si il putem schimba cum vrem. Textele in limbajul C++ au la inceput si la final semnele "

return 0;

Reprezinta valoarea pe care o returneaza functia main, adica 0. O sa vorbim mai amanuntit la capitolul 'Subprograme'.

}

Aceasta paranteza marcheaza sfarsitul functiei main.

// Afisare: Hello world
