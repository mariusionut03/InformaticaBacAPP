1. Tipul int

Permite memorarea unei valori intregi (pozitiva sau negativa).
Ea ocupa 4 octeti, deci poate memora valori din intervalul [-231, 231-1], adica [−2.147.483.648, 2.147.483.647].

int n = 100;

Daca in datele unei probleme, numere intregi depasesc 2.000.000.000 folosim tipul long long.

2. Tipurile float si double

Memoreaza numere reale. Separatorul zecimal este punctul ".".
Tipul float se reprezinta pe 4 octeti.
Tipul double se reprezinta pe 8 octeti.

float p = 3.14, r = 2.5;
double A = p * r * r;

Datele reale pot fi scrise in forma stiintifica:

double x = 1.24E+07;
// x are valoarea 1.24 * 107.

3. Tipul char

Memoreaza codul ASCII al unui caracter. Delimitarea este facuta prin " ' ". Se reprezinta pe 1 octet.

char c = 'A'; // In variabila c nu se retine caracterul 'A', ci codul ASCII al caracterului, care este 65.

4. Tipul pointer

Acest tip de data retine o adresa din memorie. Spre exemplu, adresa unei variabile.

5. Tipul bool

Tipul bool poate retine doar 2 valori: adevarat sau fals. El se reprezinta pe 1 octet. Valorile numerice sunt 1(Adevarat) si 0(Fals).

bool p = false;

6. Tipul void

Cuvantul void inseamna nimic. Aceste date nu au valori si nu poti face operatii cu ele. Le folosim doar la functii si pointeri.

Modificatori de tip

Permit schimbarea modului in care se reprezinta o data.

Ele sunt:
- signed
- unsigned
- short
- long
Pot fi aplicate la:
- int
- double
- char

Tipul Spatiu Domeniu
signed int 4 octeți cu semn [-231, 231-1], adica [-2.147.483.648, 2.147.483.647]
unsigned int 4 octeți fără semn [0, 232-1], adica [0, 4.294.967.295]
long 4 octeți cu semn La fel ca int. Echivalent cu long int.
unsigned long 4 octeți fără semn Echivalent cu unsigned int / unsigned long int.
short 2 octeți cu semn [-215, 215-1], adica [−32.768, 32.767]
unsigned short 2 octeți fără semn [0, 216-1], adica [0, 65.535]
long long 8 octeți cu semn [-263, 263-1], echivalent cu long long int.
unsigned long long 8 octeți fără semn [0, 264-1], echivalent cu unsigned long long int.
signed char 1 octet cu semn [-27, 27-1], adica [-128, 127].
unsigned char 1 octet fără semn [0, 28-1], adica [0, 255].
long double 10, 12, 16 Memorează numere reale mari.
