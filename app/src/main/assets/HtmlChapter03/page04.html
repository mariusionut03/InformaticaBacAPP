Intrari/Iesiri in C++

Operațiile de de intrare/ieșire sunt operațiile prin care un program primește date sau afișează rezultate.

1. Operatii de intrare: datele intra in program, programul citeste date

2. Operatii de iesire: datele ies din program, programul afiseaza date


Limbajul C++ oferă o modalitate uniformă de a realiza operațiile de intrare/ieșire. Este vorba despre stream sau flux. Stream-ul poate fi privit ca o înșiruire de caractere care sunt trimise într-o ordine bine determinată de la o sursă la o destinație. Programul va insera caractere în stream (dacă este un stream de ieșire, care afișează date) sau va extrage caractere din stream (dacă este un stream de intrare, din care se citesc date).


Noi vom utiliza, din biblioteca standard C++, comenzile:

cin - stream standard te intrare
cout - stream standard de iesire

Tastatura -> cin -> Program -> cout -> Ecran

Stream-ul de ieșire cout

Pentru afisatul pe ecran, o sa folosim stream-ul cout. El foloseste operatorul de insertie <<, urmat de ce se va afisa.

cout << 1234; // Afisare: 1234
cout << "Salutare"; // Afisare: Salutare
cout << a; // Afisare: (valoarea variabilei a)

Putem afișa mai multe valori în aceeași instrucțiune:

int varsta_necesara = 18;
cout << "Varsta necesara este de " << varsta_necesara << " ani.";
// Afisare: Varsta necesara este de 18 ani.

Cout nu adaugă automat sfârșit de linie sau spatii. De exemplu:

cout << "Aceasta este";
cout << "o propozitie.";
//Afiseaza: Aceasta esteo propozitie."

Trebuie sa punem noi spatiul:

cout << "Aceasta este "; //Spatiu dupa 'este'
cout << "o propozitie.";
//Afiseaza: Aceasta este o propozitie."

Dacă vrem să afișăm pe linii diferite, adaugam '\n' pentru a trece pe linia urmatoare:

cout << "Aceasta este o\n";
cout << "propozitie mai lunga!";

Sau putem folosi manipulatorul endl:

cout << "Aceasta este o" << endl;
cout << "propozitie mai lunga!";

Stream-ul de intrare cin

Cin se folosește împreună cu operatorul de extragere >>, urmat de variabila în care se va memora valoarea extrasă (variabila care se va citi). In cele mai multe cazuri, dispozitivul de intrare este tastatura.

int n;
cin >> n;

Mai întâi se declară variabila n, apoi se citește o valoare pentru ea – se extrage din cin o valoare care se memorează în variabila n. La execuție, programul așteaptă să se introducă o valoare de la tastatură. Caracterele introduse sunt transmise programului numai când se apasă tasta ENTER.

#include <iostream>
using namespace std;
int main()
{
  int n;
  cin >> n;
  cout << "n este " << n << endl;
  cout << "patratul lui n este " << n * n << endl;
  return 0;
}
// Daca citim numarul 5 pentru variabila n, se va afisa:
// patratul lui n este 25

Se pot citi valorile a două sau mai multe variabile în aceeași instrucțiune:

cin >> x >> y;

este echivalent cu

cin >> x;
cin >> y;
