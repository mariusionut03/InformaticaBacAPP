Variabile si constante C++

Orice program prelucreaza date. Datele pot fi variabile(valorile lor pot fi modificate) sau constante(valorile nu pot fi modificate).

Variabile

O variabilă reprezintă o locație de memorie unde se află o valoare de un anumit tip.
În C++, variabilele trebuie declarate, precizând tipul și identificatorul. Sintaxa este:

Tip_data Lista_Identificatori;

unde Tip_data poate fi orice tip de date in C++ (vezi pagina anterioara), iar Lista_identificatori se face o lista cu unul sau mai multe nume, ce ne vor reprezenta variabilele.

int a, x;
// Am declarat doua variabile fara valori.

int a = 1, x=6;
// Am declarat doua variabile cu valori.

Urmatorii identificatori C++ sunt corecti: a, numar, Numar, alt_numar, a2b, _suma, un_nume_de_variabila_foarte_lung.

Urmatorii identificatori C++ sunt incorecti: 2a (Incepe cu cifra), alt numar (contine spatiu), un-numar (contine minus), număr (contine diacritice, identificatorii pot contine numai litere ASCII).

Constante

Constantele sunt date care nu isi modifica valoarea in timpul executiei programului.

Ele se declara prin doua moduri:

// Prin directiva define:
#define ABC 123
// Prin modificatorul const:
const int ABC = 123;

La declararea constantelor este obligatorie initializarea.


O categorie aparte de caractere constă în secvențele ESCAPE. O secvență escape este alcătuită din două caractere, dintre care primul este backslash: \. Reprezintă caractere care nu pot fi scrise ca atare sau au un înțeles special. Din punct de vedere sintactic sunt caractere, fiind delimitate prin aprostrof. Probabil cea mai cunoscută secvență escape este '\n' – new line (enter).

'\n' – Newline
'\b' – Backspace
'\f' – Form feed
'\r' – Return
'\t' – TAB orizontal
'\\' – Backslash
'\'' – Apostrof
'\"' – Ghilimele
'\?' – Semn de întrebare
'\0' -Caracterul nul
