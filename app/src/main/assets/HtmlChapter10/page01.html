Structuri de date neomogene în C++

Cateodata avem nevoie de o sortare mai eficienta a datelor intr-un program. Spre exemplu, daca vrem sa facem un program ce reprezinta o lista de persoane, avem nevoie de mai multe date ce sunt legate intre ele:
-nume
-prenume
-localitate
-varsta
-salariu
-etc.
Pentru aceste situatii, s-a creat tipul 'struct'.

Declararea unei structuri

struct [NUME_STRUCTURĂ]
{
[TIP_1] [NUME_CAMP_1], [NUME_CAMP_2], ... [NUME_CAMP_N];
[TIP_2] [NUME_CAMP_1], [NUME_CAMP_2], ... [NUME_CAMP_N];
[...]
[TIP_N] [NUME_CAMP_1], [NUME_CAMP_2], ... [NUME_CAMP_N];
} [LISTA VARIABILE];

Cel mai probabil nu intelegi nimic din lista de mai sus, asa ca o sa dam exemplul cu numerele de telefon de mai sus:

struct Persoane
{
char nume[21], prenume[21], localitate[21];
int varsta;
double salariu;
} A, B, v[100];

Campul [LISTA VARIABILE] poate fi lasat liber, iar declararea variabilelor se poate face mai tarziu in program astfel:

Persoane A, B;
// sau
struct Persoane A, B;

Accesarea datelor

Observati ca in lista de variabile am adaugat un vector, v[100]. Astfel, putem retine in structura Persoane cel mult 100 de persoane astfel:

cin >> v[1].varsta; // Introducem de la tastatura varsta primei persoane;
cout << v[20].nume; // Afisam numele celei de-a 20-a persoana;
cout << v[20].nume[0]; // Prima litera din numele celei de-a 20-a persoana;
cin >> v[100].localitate[0]; // Citim prima litera din numele localitatii ultimei persoane;
