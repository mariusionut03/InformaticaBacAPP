Recursivitatea

Recursivitatea reprezintă proprietatea unor noțiuni de a se defini prin ele însele.

Să scriem o funcție C++ care returnează factorialul unui număr natural transmis ca parametru. Varianta nerecursivă (iterativă) este următoarea:

int fact(int n)
{
  int p = 1;
  for(int i = 1 ; i <= n ; i ++)
    p = p * i;
  return p;
}

O variantă recursivă pentru determinarea lui n!:

int fact(int n)
{
  if(n == 0)
    return 1;
  else
    return n * fact(n-1);
}

Cum functioneaza recursivitatea?

Chemam subprogramul 'fact' de mai sus cu n = 4.

Dupa verificarea valorii lui 4, va trebui sa returnam 4 * fact(3).

Astfel, calculam fact(3) si va trebui sa returnam 3 * fact(2).

Dupa, calculam fact(2) si va trebui returnat 2 * fact(1).

Dupa, calculam fact(1) si va trebui returnat 1 * fact(0).

Acum, fact(0) stim ca este 1. Acum putem face calculele.

fact(1) este 1 * 1 = 2; fact(2) este 2 * 1 = 2; fact(3) este 3*2 = 6 si fact(4) este 4 * 6 = 24.
