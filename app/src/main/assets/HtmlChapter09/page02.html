Siruri de caractere in C++

Un sir de caractere este un tablou unidimensional(vector) ce contine caractere.

Astfel, pentru reprezentarea în C/C++ a cuvântului copil, care are 5 caractere, se vor folosi 6 octeți, cu valorile: 'c', 'o', 'p', 'i', 'l', '\0'.

În reprezentarea internă, după ultimul caracter (byte, octet) valid din șir se află caracterul '\0' – caracterul cu codul ASCII 0, numit și caracter nul. Toate sirurile de caracter il contin la final, el marcand sfarsitul. Astfel, cand vrem sa reprezentam un cuvant de maxim 10 litere, vom declara vectorul cu maxim 11.

Declararea unui șir de caractere

char s[11]; // putem memora in sir maxim 10 caractere char s[11] = "copil"; // Se folosesc doar 6 caractere (5 + cel nul) char t[]="copil"; // se aloca automat 6 octeti char x[6]={'c','o','p','i','l','\0'}; char z[]={'c','o','p','i','l','\0'}; // se aloca automat 6 octeti pentru sir

Afișarea unui șir de caractere

cout << s << endl;

Citirea unui șir de caractere

cin >> s;

Nu se poate citi un sir cu spatii daca folosim operatorul '>>'.

Pentru a citi șiruri care conțin spații, putem folosi metoda 'getline'.

cin.getline(s , 11);

Un exemplu in C++:

#include <iostream> using namespace std; int main(){ char nume1[31], nume2[31]; cout << "Cum te cheama? (nume, prenume) "; cin.getline(nume1, 31); cout << "Cum il cheama pe prietenule tau? "; cin.getline(nume2 , 31); cout << "Te numesti " << nume1 << endl; cout << "Esti prieten cu " << nume2 << endl; return 0; }

Referirea unui caracter din șir

Deoarece șirurile de caractere sunt de fapt tablouri, pentru referirea unui caracter din șir se folosește operatorul [], ca în exemplul următor:

char s[]="abac"; // sirul consta din 5 caractere: cele 4 litere si caracterul nul '\0' cout << s[3]; // c s[1] = 'r'; cout << s; // arac

Parcurgerea unui sir de caractere

Sunt unele probleme unde trebuie sa parcurgem tot sirul de caractere pentru a verifica anumite caractere sau pentru a edita fiecare caracter in parte. Avem momente cand nu stim cate caractere contine sirul. Mai jos avem doua variante:

char s[11]; cin.getline(s); int i = 0; while(s[i] != '\0') { // Instructiuni asupra caracterului de pe pozitia i; i ++; }

Sau, putem folosi functia 'strlen', care returneaza lungimea sirului:

#include <iostream>; #include <cstring>; [...] // Includem la inceputul programului libraria 'cstring' pentru a putea accesa comanda 'strlen' for(int i = 0; i < strlen(s); i++) { //Instrunctiuni asupra caracterului de pe pozitia i, accesat cu s[i]; }
